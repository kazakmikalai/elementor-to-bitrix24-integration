# Elementor Pro to Bitrix24 Lead Integration (No Plugins)

Легкий и быстрый PHP-скрипт для прямой интеграции форм **Elementor Pro** с **Битрикс24 CRM** через входящие вебхуки. Решение работает без использования тяжелых плагинов и сторонних сервисов.

## 🚀 Преимущества
- **Бесплатно навсегда:** Не требует подписок на коннекторы.
- **Высокая производительность:** Использует нативный хук Elementor и WordPress HTTP API.
- **Безопасность:** Данные передаются напрямую в вашу CRM без промежуточных серверов.
- **Гибкость:** Легко адаптировать под любые поля.

## 🛠 Требования
- WordPress с установленным **Elementor Pro**.
- Активный аккаунт **Битрикс24**.

## 📖 Инструкция по установке

### 1. Настройка Битрикс24
1. Перейдите в раздел **Разработчикам** -> **Другое** -> **Входящий вебхук**.
2. В правах доступа выберите **CRM (crm)**.
3. Сохраните и скопируйте URL. Он будет выглядеть так:  
   `https://your-domain.bitrix24.ru/rest/ID/TOKEN/`
4. Добавьте к ссылке метод: `crm.lead.add.json`.

### 2. Настройка формы Elementor
1. В редакторе Elementor выберите вашу форму.
2. В настройках полей (вкладка **Advanced**) задайте ID:
   - Для имени: `Name`
   - Для телефона: `Phone`
3. В настройках формы (вкладка **Content** -> **Form Settings**) убедитесь, что форма имеет уникальное имя или просто используйте универсальный скрипт.
4. В **Actions After Submit** оставьте только `Collect Submissions` или `Email`. **Webhook выбирать не нужно**.

### 3. Добавление кода
Добавьте содержимое файла `functions.php` из этого репозитория в файл `functions.php` вашей темы или воспользуйтесь плагином **Code Snippets**.

## ⚙️ Как это работает
Скрипт перехватывает событие отправки формы (`elementor_pro/forms/new_record`), собирает данные методом `get_formatted_data()`, очищает номер телефона от лишних символов и отправляет структурированный POST-запрос на ваш вебхук Битрикс24.
